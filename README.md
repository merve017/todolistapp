# To-Do App

The To-Do App is based on Flutter and Firebase/firestore and developed by Carlo Spinello, Merve Taskaya and Tanja Wechselberger.


## Getting Started
- Clone the repo
- Install the dependencies
- Run it


## Preview 
The App is working as a iOS App, Android App and Web App.
A Live Preview of the Web App can be found on: 
- [To-Do App](https://fh-todolistapp.web.app/)
